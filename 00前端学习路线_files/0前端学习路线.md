## 5.准备简历  
- 雷点：校园经历不用写（班干部学生会之类的），个人总结不用写  
- 个人技能：卢前端相关的技能，比如熟悉html、css、JS等等  
- 项目经历：找人包装一下，然后写在简历上，可分为技术栈、项目介绍、项目亮点
> eg: SaaS流程管理系统
> Vue3 + Vue-Router + Pinia+Webpack+Vue Flow
> 2025.4-2025.5
> 项目介绍：流程管理SaaS平台，包含流程设计器、任务审批、数据看板等模块。支持通过拖拽方式配置审批节点，实现采购、请假等业务流程线上化。
> 项目亮点：
> - 基于 Composition API 实现响应式流程设计器，使用第三方库vue flow实现节点拖拽与连线。  
> - 基于 Vue3 + JavaScript 搭建模块化工程，通过Vue Router 实现路由懒加载与动态权限拦截。  
> - 重构状态管理架构，将Vuex 迁移至 Pinia，借助其强类型支持和模块化设计，提升代码可维护性并减少模板代码量。  
> - 通过图片压缩，使用CDN、Webpack分包、动态导入、优化代码拆分等常规性能优化手段将 FCP 指标由3.7s优化成1.8s.  

- 教育背景：你的学校、获得的竞赛奖项、绩点
